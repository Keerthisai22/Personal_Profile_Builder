# Personal Profile Builder (Python)

A simple real-time Python project that collects user details and displays a clean, structured personal profile.

## Features
- Takes user input in real time
- Displays profile in a professional format
- Beginner-friendly and easy to understand

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program using:
   python personal_profile_builder.py
